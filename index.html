- - - - - - - - Descriptions to all HTML Tags - - - - - - - -

Basic Structure Tags

  • !DOCTYPE tag :- Declares the document type(HTML5).
  • HTML tag :- Root of the HTML document.
  • HEAD tag :- Contains metadata (not visible on the page).
  • TITLE tag :- Sets the title shown in the browser tab.
  • BODY :- Holds all the visible content of the webpage.

Text Formatting Tags

  • Heading Tags :- H1 to H6 - h1 is the largest
  • P tag :- Paragraph.
  • br tag :- Line break.
  • hr tag:- Horizontal line (divider)
  • strong tag :- Important text (bold).
  • em tag :- Emphasized text (italic).
  • b tag :- Bold (without semantic importance).
  • i tag :- Italic (without semantic importance).
  • u tag :- Underlined text.
  • Small tag :- Smaller text.

Links & Media Tags

  • a tag :- Anchor tag for hyperlinks
  • img tag :- Image. Requires src and alt.
  • Video and Audio tag :- Embed media files with playback controls.

Tables

  • table tag :- Defines a table.
  • tr tag :- Table row.
  • th tag :- Table header.
  • td tag :- Table data cell.

Media & Embeds

  • source tag :- Specifies media resources (used in video, audio, or picture).
  • track tag :- Subtitles or captions for video/audio.
  • canvas tag :- A space for drawing graphics via JavaScript.
  • svg tag :- Scalable Vector Graphics (inline graphics)
  • iframe tag :- Embeds another HTML page inside the current one.
  • embed tag :- Embeds external content (e.g. PDFs, Flash).
  • object tag :-Used to embed objects like media or plugins.
  • param tag :- Parameters for object tag.

Form Elements Tags

  • fieldset tag :- Groups form elements.
  • legned tag :- Caption/title for a fieldset.
  • datalist tag :- Predefined options for an input (auto-suggest).
  • output tag :- Displays the result of a calculation.
  • progress tag :- Shows progress of a task (e.g., loading bar).
  • meter tag :- Displays a scalar measurement (e.g., battery level).

Semantic HTML5

  • figure and figcaption tags :- For images with captions.
  • mark tag :- Highlighted text.
  • code tag :- Code snippets.
  • pre tag :- Preformatted text (preserves spacing).
  • time tag :- Time or date.

Lists Tags

  • ul tag :- Unordered list (bullets).
  • ol tag :- Ordered list (numbers).
  • li tag :- List item.

Obsolete or Rare Tags

  • center tag :-
  • font tag :- Set font styles (use CSS instead).
  • marquee tag :- Creates scrolling text (fun, but outdated).
  • blink tag :- Blinking text (not supported anymore).

  • abbr tag :- Abbreviation with a tooltip.
  • address tag :- Contact info for the author/owner.
  • cite tag :- Citation or reference.
  • dfn tag :- Defines a term.
  • kbd tag :- Keyboard input (like "Press Ctrl + S").
  • samp tag :- Sample output from a program.
  • var tag :- A variable in a formula or code.

Forms (User Input)

  • form tag :- Form container.
  • input tag :- Input field (text, email, checkbox, etc.).
  • Label tag :- Label for an input.
  • textarea tag :- Multi-line text input.
  • select tag :- Dropdown menu.
  • option tag :- Option in a dropdown.
  • button tag :- Clickable button.

Layout and Structure

  • div tag :- Generic block container.
  • span tag :- Generic inline container.
  • Section tag :- Defines sections of content.
  • article tag :- Independent content (e.g., blog post).
  • header and footer tag :- Top and bottom of a section or page.
  • nav tag :- Navigation links.
  • main tag :- Main content area.
  • aside tag :- Sidebar or related content.

Metadata & Head

  • meta tag :- Provides metadata like character set, author, viewport, etc.
  • link tag :- Links external resources (like CSS files).
  • style tag :- Adds internal CSS styles.
  • base tag :- Sets base URL for all relative URLs in the document.
  • script tag :- Includes JavaScript code or files.
  • noscript tag :- Fallback content for browsers with JavaScript disabled.

Interactive Tags

  • details tag :- Toggleable content section (like FAQs).
  • summary tags :- Visible label for details (click to expand/collapse).
  • dialog tag :- Used to create modal dialogs.
  • menu tag :- Represents a list of commands.
  • menuitem tag :- Command within a menu (deprecated but was used with context menus).
